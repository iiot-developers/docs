# IIOT 憑證變數設定

憑證設計有自訂變數的功能

## 基本變數

憑證連結
```
{{codeLink}}
```
憑證碼

```
{{code}}
```

## 訂單自訂變數

可於上傳訂單中，新增自訂變數

**限制: 僅支援小寫、可用底線分隔**

範例：

在Excel新增一個欄位，標頭為`shipment_address`

即可在憑證內容中，使用自訂變數
```
您的宅配地址為：{{order.shipment_address}}
```
## 範例

```
親愛的 {{order.name}} 您好，
您的預約編號 {{order.order_no}} 已送達櫃檯，
請點擊連結 {{codeLink}} 至櫃檯取貨，謝謝您。
```
